# CBR API
Unofficial codebreaker.xyz API (basically a wrapper that scrapes the HTML and parses it)

## Available Scripts

In the project directory, you can run:

### `npm run dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm start`

For production mode

### `npm run test`

Run the test cases.
